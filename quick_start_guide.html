Best Support System Quick Start Guide

This document show you how to install best support system

Step by Step Install Guide

  1. After download you will get a file name support_system.zip
  2. Unzip the support_system.zip. You will get a folder installer & index.php.
  3. Upload installer & index.php into your install directory.
  4. Browse the directory from web browser ( Firefoz, Chrome,etc).
  5. Installer: Step 1, It will show you the requirement for best support system. If all requiment passed then press next
  6. Installer: Step 2, In this step put your database information and press the button next.
  7. Installer: Step 3, In this step, put your support system name & admin user information and press the button next.
  8. Installer: Step 4, In this step, put the envato purchase key here. In the left bottom corner you will the instuction how you will get purchse key from envato. If you want then you can skip this step. Keep blank purchase key to skip this step and press next.
  9. Installer: Final Step, In this step, you don't need to do anything after process you will see to link first one is admin login link and second one is site link. Admin link will be "your site url"/admin.
